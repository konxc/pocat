## Quick Start

Get up and running with Pocat in minutes:

```bash
# Download any YouTube video
curl -X POST https://api.pocat.io/v2/projects \
  -H "Content-Type: application/json" \
  -d '{
    "title": "My Video Project",
    "youtubeUrl": "https://www.youtube.com/watch?v=VIDEO_ID",
    "userId": 1,
    "downloader": "yt-dlp"
  }'
```

Pocat is built with modern technologies:

- **Backend**: AdonisJS (Node.js/TypeScript)
- **Database**: SQLite (development) / Turso (production)
- **Video Processing**: FFmpeg with multiple downloader engines
- **Deployment**: Docker-ready with CI/CD support
