# Welcome to Pocat

Pocat is a powerful AI Video Clipper API that transforms long YouTube videos into engaging clips with multiple download options and quality selections.

## What is Pocat?

Pocat provides a robust backend API for:
- **YouTube video downloading** with multiple engines
- **Quality selection** from 144p to 4K
- **Metadata extraction** and processing
- **AI-ready video processing** for clip generation
